A Current-Meter-based clock.  

Inspired by the Adafruit clock:


 https://learn.adafruit.com/trinket-powered-analog-meter-clock/overview

Notes about UI -
There is one control, the rotary encode with a button.
Events - 
3s Press
200ms press - Switch between M & H 
Rotation

Press for 3s - That should start a needle dance - R->L->R and take you to setting the minutes.  

States -
Idle
Dance 1
Dance 2
Dance 3 
Set Minutes 
Set Hours
